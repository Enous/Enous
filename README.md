## One Cikan Proje: PulsePilot Automation Studio
Python ile yazilmis modern bir otomasyon uygulama iskeleti:
- Zamanlanmis gorev calistirma (scheduler)
- URL saglik kontrolu
- Klasor yedekleme (zip)
- Eski dosya temizleme
- Komut calistirma + JSONL log
